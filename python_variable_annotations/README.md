# Python - Variable Annotations

This project introduces type annotations in Python.  
You will learn how to add type hints to variables, functions, and return values to make your code more readable and maintainable.

## Learning Objectives

- Understand type annotations in Python 3
- Use type hints for variables and function parameters
- Use type hints for function return values
- Use features from the `typing` module such as `List`, `Tuple`, `Dict`, and `Union`
- Understand duck typing and type checking

## Requirements

- Python 3.7+
- All files should be executable
- Code should follow `pycodestyle` (version 2.5)
- All modules and functions must have type annotations
- All files must end with a new line

## Files

| File | Description |
|------|-------------|
| `0-add.py` | Function that adds two floats |
| `1-concat.py` | Concatenates two strings |
